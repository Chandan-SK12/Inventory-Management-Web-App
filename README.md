# Inventory Management Web App

## Overview
Developed a full-stack backend inventory management application using Java, Spring Boot, and MySQL. Exposed RESTful APIs for secure inventory access.

## Tech Stack
- Java
- Spring Boot
- MySQL
- Postman (for testing)

## Features
- Role-based access control
- CRUD API endpoints
- Secure login and user authentication

## Project Structure
Standard Spring Boot folder structure with `Controller`, `Service`, and `Repository` layers.
